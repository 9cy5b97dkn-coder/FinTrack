# FinTrack - Личный финансовый трекер

> Красивое автономное PWA-приложение для учета расходов, доходов, вкладов и подписок. Все данные хранятся локально на устройстве.

## Возможности

- **Красивый интерфейс** — современный дизайн с тёмным хедером, плавающими карточками и плавными анимациями
- **Без авторизации** — приложение открывается сразу, никаких регистраций
- **Автономная работа** — работает без интернета, все данные в LocalStorage
- **PWA** — устанавливается на iPhone/Android как нативное приложение
- **22 категории расходов** — цветные круги с эмодзи в стиле CoinKeeper
- **8 категорий доходов** — зарплата, фриланс, инвестиции, кэшбэк и др.
- **Счета** — неограниченное количество счетов с иконками и цветами
- **Вклады** — отслеживание сроков с обратным отсчётом дней
- **Подписки** — ежемесячные/еженедельные/ежегодные платежи
- **Бюджет** — лимиты по категориям с цветными прогресс-барами
- **Аналитика** — графики расходов по месяцам и категориям, советы
- **CSV Импорт** — загрузка выписок из Сбербанка, Тинькофф, ВТБ, Альфа-Банка
- **Экспорт/Импорт JSON** — резервное копирование всех данных

## Быстрый старт

### Установка на iPhone

1. Откройте [приложение](https://9cy5b97dkn-coder.github.io/FinTrack/) в **Safari**
2. Нажмите **"Поделиться"** (квадрат со стрелкой внизу)
3. Прокрутите вниз и выберите **"На экран «Домой»"**
4. Нажмите **"Добавить"**

### Установка на Android

1. Откройте [приложение](https://9cy5b97dkn-coder.github.io/FinTrack/) в **Chrome**
2. Нажмите **меню (три точки)** → **"Установить приложение"**
3. Подтвердите установку

## CSV Импорт из банков

| Банк | Как получить CSV |
|------|-----------------|
| **Сбербанк** | Выписка → Сохранить в CSV |
| **Тинькофф** | История → Экспорт → CSV |
| **ВТБ** | Выписка → Скачать → CSV |
| **Альфа-Банк** | Операции → Экспорт → CSV |

## Технологии

- **HTML5** — семантическая разметка
- **CSS3** — кастомные свойства, гриды, флексы, анимации
- **Vanilla JavaScript** — без фреймворков, чистый JS
- **Chart.js** — графики и диаграммы (подгружается с CDN)
- **LocalStorage API** — хранение данных на устройстве
- **PWA** — Progressive Web App, работает офлайн

## Данные и приватность

- **Все данные хранятся локально** на вашем устройстве через LocalStorage
- **Ничего не отправляется** на сервер — полная приватность
- **Экспортируйте JSON** для резервного копирования
- **Импортируйте JSON** для восстановления данных
